## 4. 变现跟踪

### 4.1 Google Analytics 事件设置

| 事件名称 | 触发条件 | 追踪目的 |
|---------|---------|---------|
| tool_click | 点击工具链接 | 热门工具分析 |
| category_view | 浏览分类页 | 用户兴趣分析 |
| search_use | 使用搜索功能 | 搜索意图分析 |
| affiliate_click | 点击联盟链接 | 联盟转化追踪 |
| newsletter_signup | 订阅 Newsletter | 邮件营销转化 |

**GA4 事件代码示例：**
```javascript
// 联盟链接点击
gtag('event', 'affiliate_click', {
  'event_category': 'engagement',
  'event_label': 'tool_name',
  'value': 1
});
```

### 4.2 Affiliate 链接点击追踪方案

| 联盟平台 | 追踪方式 | UTM 参数 |
|---------|---------|---------|
| Amazon | Amazon Associates + UTM | source=amazon&medium=affiliate |
| ClickBank | 自定义追踪链接 | source=clickbank&campaign=ai_tools |
| Impact | ImpactRadius 追踪 | 使用平台自带追踪 |
| 工具官网 | 短链+UTM | source=tool_name&medium=affiliate |

**推荐 UTM 结构：**
```
https://example.com?utm_source=ai_tools_navigator&utm_medium=affiliate&utm_campaign=tool_name
```

**推荐工具：**
- 免费：Bitly（基础追踪）、Pretty Links
- 付费：ThirstyAffiliates（WordPress）
