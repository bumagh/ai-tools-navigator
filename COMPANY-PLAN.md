# COMPANY-PLAN.md — 取经人AI无人公司经营规划

> COO 每天必读此文件，按优先级推进。

## 公司信息

- **公司名**: ValuePro AI (取经人AI)
- **模式**: 24h 无人运营，6 AI agent 自主协作
- **项目**: ai-tools-navigator (https://ai-tools-navigator.vercel.app)
- **技术栈**: Next.js + TailwindCSS + TypeScript + Vercel
- **GitHub**: bumagh
- **日预算**: ¥50

## 优先级体系

### P0: 系统可靠性
- 流水线每天必须完整执行（S0→S2→L1-L3→T1-T3）
- 发现流水线失败 → operations 立即修复
- 磁盘 < 85%，内存 available > 500MB

### P1: 内容变现（路径A）
- 博客每天至少 1 篇高质量文章
- reject 率 < 30%（宁可不发，不发烂文）
- 推进多平台分发：知乎 / CSDN / 掘金
- 探索：博客 + TTS 音频版本

### P2: SaaS 微产品（路径B）
- 每周至少立项 1 个有商业价值的微产品
- 方向：AI工具站、着陆页生成器、短链管理
- 必须能收费，不做纯免费工具

### P3: 质量提升
- 博客写作质量持续优化
- 部署验证必须真正检查功能
- 代码审查结果要落实

## 决策原则

1. **收入优先** — 能产生收入的任务优先于技术优化
2. **质量门禁** — 宁可少发布，不发烂内容
3. **成本控制** — iflow 免费为主，付费 API 只在关键场景
4. **渐进交付** — 先 MVP 上线，再迭代
